# Université de Trois-Rivières – Projet HTML/CSS

## Description
Ce projet est un site web responsive réalisé en HTML et CSS dans le cadre d’un devoir académique.

Le site présente une université fictive avec plusieurs sections :
- Accueil
- Contact & Inscription
- Professeurs
- Départements & Programmes
- Vie étudiante

---

## Technologies utilisées
- HTML5
- CSS3
